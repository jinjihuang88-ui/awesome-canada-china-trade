# Contribution Guidelines / 贡献指南

Thank you for helping make this list more awesome!
感谢你帮助完善这份精选列表！

## Adding an Entry / 添加条目

Please ensure your pull request adheres to the following guidelines:

- Search previous suggestions before making a new one to avoid duplicates.
- Make sure your suggestion is genuinely useful and relevant to Canada-China B2B trade.
- The suggestion must be in active use and not abandoned/deprecated.
- Each entry must include a bilingual description (Chinese and English).
- Make an individual pull request for each suggestion.
- Use the following format: `- **[Name](link)** — 中文描述。*English description.*`
- Link additions should be added in alphabetical order within each category.
- New categories, or improvements to the existing categorization, are welcome.
- Check your spelling and grammar.
- Make sure your text editor is set to remove trailing whitespace.
- The pull request and commit should have a useful title.

---

请确保您的 Pull Request 符合以下规范：

- 提交前请先搜索现有条目，避免重复。
- 确保建议的资源真实有用，与加中 B2B 贸易直接相关。
- 资源需处于活跃维护状态，不接受废弃或已停止维护的项目。
- 每个条目必须包含中英双语描述。
- 每个建议单独提交一个 Pull Request。
- 格式：`- **[名称](链接)** — 中文描述。*English description.*`
- 在各分类内按字母顺序排列。
- 欢迎提议新分类或改进现有分类结构。

Thank you for your contribution! / 感谢您的贡献！
